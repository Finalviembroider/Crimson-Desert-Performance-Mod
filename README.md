## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Mods don't apply | Run as Administrator, verify game install path |
| FPS still capped | Disable in-game V-Sync + use FPS Unlocker |
| Texture pop-in | Increase Texture Streamer buffer size, ensure SSD |
| Shader stuttering | Run Shader Cache pre-compile before first session |
| Foliage looks too sparse | Increase Foliage Reducer density from 25% to 50% |
| Crashes in cities | Lower Draw Distance for NPCs, increase Memory Manager pool |
